O programu

Program Athletics pro totální stanice Topcon a Sokkia

Podporované řady totálních stanic :

  • AT : Topcon GPT, QS
  • AT-PS : Topcon PS, OS - Sokkia SX - 320 x 240
  • AT-GT : Topcon GT - Sokkia IX - 640 x 480

Program nabízí :

  • Měření atletických disciplín :
    • skok daleký, trojskok
    • hod oštěpem
    • hod diskem
    • hod kladivem
    • vrh koulí
    • skok o tyči, skok vysoký
  • Možnost online exportu dat s volbou výstupu
  • Ukládání výsledků do protokolů
  • Grafické zobrazení výsledků
